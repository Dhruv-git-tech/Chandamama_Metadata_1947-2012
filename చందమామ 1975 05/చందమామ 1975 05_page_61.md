# వ్యారీవారి "సమయోచిత పాటీలో సందర్భాన్ని చెప్పండి"

2. మొదటి బహుమానాలు — మీ ఇష్టమువచ్చిన ఆభరణాలసమితి ఒక్కొక్కటి — విలువ రూ. 10,000/-

2. రెండవ బహుమానాలు. రెఫిజిరేటరుగాని వంట సరంజాముగాని ఒక్కొక్కటి — విలువ రూ. 5,000/-

50 కన్సోలేషన్ బహుమానాలు — కాంచీపురము చీరలు ఒక్కొక్కటి — విలువ రూ. 400/-
