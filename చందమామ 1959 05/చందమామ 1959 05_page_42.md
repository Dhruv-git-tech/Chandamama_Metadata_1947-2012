కాచి వేచియు రాని - కారణమ్మేమి
చూచివచ్చెదనంచు - చొఱువతో వచ్చె.
పచ్చి చూచునుగదా - వాడిట్టు లుండె
ఆచ్చరిత్రము చెప్పె - ఆవురుమంచు.

సువర్ణసిద్ధి :

పలదంటె వింటివా - వచ్చితి విటకు
కలిగిన బంగారు - కాల దన్నితివి.
చదువున్ననేమి సం-స్కారము నేల?
మొదల కావలె తెల్వి - అది వినలేదె ?

చక్రధారి :

ఆది యేమి ఎటువలె - నది వినిపించు

సువర్ణసిద్ధి :

ఆదె వినిపింతు, ఉన్నది కథ యొకటి.

నలుగురు సోదరుల్ - కలరొక యూర
పలు మంత్రశాస్త్రాల - పండితుల్ ముగురు
కడగొట్టువాని కొక్కటి రాదుగాని
మిడిదేరినది తెల్వి - మెలకువై నతను.
తమ విద్య చూపి ధనము దెత్తుమంచు
తమ రాజధానికి - తరలిరి వారు.
చిన్నవాడును వచ్చి - చేరగా చూచి
అన్నలిద్దరు వాని - నాగుమన్నారు.
చదువేమి రాని యీ - చపట యేమిటిక?
వదిలించు తమ డబ్బు - వద్దనినారు.

మూడవ అన్న త-మ్ముని బుజ్జగించి
వీడు తెల్విగలాడు - వెంట రానిండు.
ఆని చెప్ప నందఱు - చనుచున్నవారు
వనములు వాగులు - వరుసగా దాటి.
ఆడవిని బడిపోవ - నట కానపచ్చె
బడుగు సింహపు ఆస్టి-పంజర మొకటి
మంత్రా లెతీంగిన - మానవుల్ పూని
మంత్రవిద్యను చూప - మతి జనియించి
ఎముకల కూత్చి - తమ మంత్రశక్తి
నిముసాన బ్రదికింప - నెంచిరి దాని.
పసివాడు నవ్వి ఆ-పని వలదనెను
అసమర్థుడవు చూడు - మనిరి వారేమొ.
