# మన దేశభాషలు నేర్చుకుందాం

ఈ జాబితాలో మొదట మీకు తెలిసిన భాషలలోని పదాలను చదవండి.

କ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରିକ୍ରି
