# ఆమెరోవాళ్లు

త్యాజ్యం న ధైర్యం విదురేపి కాలే,
ధైర్యాత్ కదాచిత్ గతి మాప్పుయాత్ సః,
యథా సముద్రేపి చ పోతభంగే
సాం యాత్రికో వాంఛతి తర్తు మేవ.

1

[ కష్టకాలంలో కూడా ధైర్యం విదువరాదు. ధైర్యం ఉంటే ఒకప్పుడు తప్పించుకునే మార్గం దొరకవచ్చు. సముద్రం మీద ఒడలు పగిలితే యాత్రికుడు తప్పించుకోవటానికే చూస్తాడు. ]

వ్యసనే మిత్రపరీక్షా,
శూరపరీక్షా రణాంగనే భవతి,
వినయే భృత్యపరీక్షా,
దానపరీక్షా చ దుర్భిక్షే.

2

[ కష్టకాలంలో మిత్రపరీక్షా, యుద్ధభూమిలో శూరపరీక్షా, వినయంలో సేవకుడిపరీక్షా, శరువులో దానపరీక్షా జరుగుతాయి. ]

న సదశ్యః కశామాతం,
న సింహా ఘనగర్ణితం,
పరై రంగులినిర్దిష్టం,
న సహంతే మనస్వినః.

3

[ మంచి గుర్రాలు దెబ్బపడవు. సింహాలు మేమగర్జన నహించవు. స్వాభిమానం గలవాడు మరొకుడి చేత వేలెత్తి చూపించబడటం నహించదు. ]
