# అమృతాంజన్

తెలనాప్పులకు, జలుబులకు, ఒళ్లునాప్పులకు ఇంటెంటా ఉండవలసిన దివ్య ఔషధము

తలనొప్పులకు, జలుబులకు, ఒళ్ళు నొప్పులకు ఉపశమనం కలిగించే అమృతాంజన్‌ను భాధగా ఉన్న బోట కొద్దిగా రాసి రుడ్డండి. దానిలో వేరి ఉన్న పది రోష్టమైన ఔషధాలు నిమిషాల మాద గుణం కలిగిస్తాయని మారు తెలుసుకొంటారు.

ఆదాపేసే పెద్ద నీసాలలోను, చిన్న నీసాలలోను తక్కువ వెలకు ఇమిడికగా చిన్న ఉద్యాలలోను లభిస్తుంది.

అమృతాంజన్
80 సంవత్సరాలకు
ప్రైలడిన నమ్మకమైన
దివ్య ఔషధము.
