# యువ

(సచిత్ర మాసపత్రిక)

విడిప్రతి వెల రు. 0-75 :: సాలుచందా రు. 9-00

## ఈ సంచికలో

గ్రహ శకలం ... కొడవటిగంటి

ఇద్దరు గొప్పవాళ్ళు ... ఇనుకపల్లి దక్షిణామూర్తి

రాగరేఖలు ... రమాకుమారి

నేనింతవాణ్ణి ఎలా అయ్యాను ... శాయన్న

చీకటి గదులు (సీరియల్) ... గోపిచంద్

స్త్రీల ప్రత్యేక శీర్షిక “యువతి లోకం”లో

గిరిక ... దాశరథి

ఇవి గాక మతములు - ప్రవక్తలు, పని లేని మంగలి, “డుంబు” వర్ణ చిత్రకథ, సినిమా క్విజ్, చిత్ర పరిచయ పోచీ మొదలగు ఎన్నో ఆకర్షణలతో

కావే మీ ఏజంటు నడగండి
