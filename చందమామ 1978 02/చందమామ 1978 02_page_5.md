# ఆమెరోవాళ్లు

న హ్యాతో ధర్మచరణం
కించి దస్తి మహత్తరమ్,
యథా పితరి శుభూషా,
తస్య వా వచనక్రియా.

1

[ తండ్రికి సేవలు చేయటమూ, తండ్రి మాట పాలించటమూ కన్న గొప్ప ధర్మం లేదు.]

అస్మాకం చ కులే పూర్వం
సగర స్యాజ్ఞయా కితువః
ఖనర్ఫి స్సాగరై రూపమి
మవాప్త స్సుమహా, న్యధః.

2

[ పూర్వం “మా” నూర్యవంశంలో నగరుడి ఆజ్ఞానుసారం భూమిని తవ్వి అతని కొడుకులు చనిపొయారు. ]

జమదగ్నేన రామేణ
రేణుకా జననీ స్వయం.
కృత్తా పరశు నారణ్యే
పితుర్వచన కారిణా.

3

[ జమదగ్ని కొడుకైన పరశురాముడు తండ్రి మాట పాటించి ఆరణ్యంలో తన తల్లి అయిన రేణుకను గొడ్డరితో నరికాదు.]
