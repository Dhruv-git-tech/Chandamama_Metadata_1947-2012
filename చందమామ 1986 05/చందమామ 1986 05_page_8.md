# చండ్రమామ

నంస్థాపకుడు : 'చ క్రపాణి'

నంచాలకుడు : నాగిరెడ్డి

ఈ నెలబేతాళకథ ["లోపం ఎవరిది?"]కి అధారం, జాన్నలగడ్డ రత్పు రచన. అధికారంలో వున్నవాడిలో దయా ధర్మగుణాలు లోపించినప్పుడు, సామాన్యుడి జీవితం నరకప్రాయమవుతుంది. అయితే, అలాంటి సామాన్యవ్యక్తి అధికారదర్శానికి తలవంచక, ధైర్యంగాదాన్ని ఎదిరించినప్పుడు సత్ఫలితాలు కలిగే అవకాశం వున్నది. ఈ సంగతి "ఉన్నత న్యాయ స్థానం" అన్న కథలో స్పష్టపరచబడింది.
