# విజ్ఞప్తి

ఈ నెల ‘చందమామ’ విడుదల ఆలస్యమైనందుకు మన్నించవల సిందిగా పాఠకులనూ, ఏజంట్లనూ కోరుతున్నాము. ఇందుకు ముఖ్య కారణం పవర్ సప్లయిలో వచ్చిన తగ్గుదల. ఈ చిక్కులను ఎదుర్కొని పత్రికను సకాలంలో వెలువరించేందుకు గట్టి ప్రయత్నాలు చేస్తున్నాము.

ఈ అనివార్య ఆలస్యాలను ఉదారతతో స్వీకరించమని ఆర్థిస్తున్నాము.
